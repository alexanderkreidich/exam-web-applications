# Электронная библиотека

Автоматизированная информационная система «Электронная библиотека»: реестр книг
и читателей, поиск книг, рецензии и разграничение прав по ролям.

Экзаменационный проект по дисциплине «Разработка веб-приложений».
Выполнил: Крейдич Александр Дмитриевич, группа 241-327. Индивидуальный
вариант 3 — функциональность поиска книг.

## Возможности

- Список книг на главной странице с сортировкой по году, средней оценкой и
  числом рецензий; пагинация по 10 книг.
- Поиск книг по названию, автору, жанрам, году и диапазону объёма; фильтр
  сохраняется при переходе по страницам.
- Просмотр книги с обложкой, Markdown-описанием и рецензиями.
- Добавление, редактирование и удаление книг с учётом прав:
  - просмотр — всем;
  - редактирование — администратор и модератор;
  - добавление и удаление — только администратор.
- Загрузка обложек с дедупликацией по MD5-хэшу.
- Рецензии с оценкой и Markdown-текстом; один пользователь — одна рецензия
  на книгу.
- Аутентификация с ролями (администратор, модератор, пользователь).

## Стек

Flask, Flask-SQLAlchemy, Flask-Migrate, Flask-Login, Bleach, Markdown, EasyMDE.
СУБД: SQLite для разработки, MySQL на хостинге.

## Установка и запуск

```bash
# Виртуальное окружение
python3 -m venv ve
. ve/bin/activate
pip install -r requirements.txt

# Все команды выполняются из каталога app/
cd app

# Применение миграций
flask db upgrade

# Заполнение справочников (роли, жанры) и создание администратора
flask seed

# Запуск
flask run
```

После `flask seed` доступна учётная запись администратора: логин `admin`,
пароль `admin`.

## Конфигурация

Параметры подключения к БД задаются в `app/config.py`. Для перехода на MySQL
нужно раскомментировать строку `SQLALCHEMY_DATABASE_URI` с
`mysql+mysqlconnector://...` и указать свои реквизиты.

## Структура

```
app/
├── app.py        # сборка приложения, отдача обложек, Markdown-фильтр
├── config.py     # конфигурация
├── models.py     # модели данных
├── auth.py       # аутентификация и проверка прав
├── books.py      # список, поиск, просмотр и CRUD книг
├── reviews.py    # рецензии
├── tools.py      # фильтр поиска, сохранение обложек, санитизация, Markdown
├── commands.py   # CLI-команда seed
├── migrations/   # миграции БД
├── templates/    # шаблоны
└── static/       # статика
```
